## ğŸ“ Project Description
This project is a **currency converter in Java**, which allows you to convert between different currencies in real time using the **ExchangeRate-API**.

The program runs from the console and dynamically queries the exchange rate between USD, ARS, BRL, and COP.

Includes:
- Interactive options menu
- API consumption using `HttpClient`
- JSON parsing using **Gson**
- Direct and reverse conversion logic
